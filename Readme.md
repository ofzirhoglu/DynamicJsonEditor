# Dinamik JSON Editörü

Dinamik JSON Editörü, kullanıcıların kolayca JSON veri yapıları oluşturmasını, düzenlemesini ve önizlemesini sağlayan modern ve etkileşimli bir web uygulamasıdır.

## Özellikler

- JSON verisini görsel olarak düzenleyin, yeni alanlar ekleyin, silin veya güncelleyin
- Her obje ve diziye (boş olanlar dahil) kolayca alan ekleyebilme
- Alan adları için anında doğrulama ve hata mesajları
- JSON verisini doğrudan yükleyebilme veya örnek veriyle başlama
- JSON çıktısını anında önizleme ve kopyalama
- Karanlık (Dark) ve aydınlık (Light) tema desteği
- JSON doğrulama ve temizleme butonları

## Kullanım

1. `index.html` dosyasını bir tarayıcıda açın.
2. JSON verinizi görsel olarak oluşturun veya yükleyin.
3. Temayı sağ üstteki buton ile değiştirebilirsiniz.
4. JSON çıktısını kopyalayabilir veya doğruluğunu kontrol edebilirsiniz.
